# Project Satsuki
2D Visual Novel with aspects of care-giving, isometric exploration, and active time battle, all while being a fan project for Touhou and a loose adaptation of Osana Reimu.

You, the caregiver named Rumia, are to care for Reimu from 1997 to 2002, supporting her growth through event scheduling, conversations, and exploration around Gensokyo. Be wary, however, as this period of transition still has lingering effects of the past. What mysteries can you uncover that'll ultimately change the projection of Reimu's future?

----
> [!IMPORTANT]
> Project Satsuki is a fan game of the Touhou Project, made by Team Shenghai Alice--ZUN--and a loose adaptation of a fan webcomic called Osana Reimu by Joyful. As such, this project is a derivative of such and does not reflect any of the author's thoughts.

# Controls
### General
`ESC` - To escape back to the main menu.

`LMB` - To navigate anywhere else.

`ENTER` - To open up the main menu's selection
